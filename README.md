# Claude–Codex Köprüsü

Bu özel depo, Hasan ile ChatGPT arasındaki konuşmalardan gelen yazılım görevlerini bilgisayardaki Claude Code'a iletmek ve sonuçları Codex ile denetlemek için kullanılır.

## Amaç

- Hasan yalnızca ChatGPT ile konuşur.
- ChatGPT görev metnini ve referans dosyalarını bu depoya koyar.
- Bilgisayardaki yerel köprü görevi alır ve Claude Code'u çalıştırır.
- Claude'un ürettiği proje Codex tarafından denetlenir.
- Gerekirse düzeltme turu otomatik başlatılır.
- Son durum ChatGPT tarafından Hasan'a bildirilir.

Bu depo özeldir. API anahtarları, parolalar ve kişisel sırlar bu depoya hiçbir zaman kaydedilmemelidir.
